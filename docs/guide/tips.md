# 操作技巧与快捷键

## Markdown 编辑技巧

**快速插入格式**

| 输入 | 效果 |
|------|------|
| `#` + 空格 | 一级标题 |
| `##` + 空格 | 二级标题 |
| `-` + 空格 | 无序列表 |
| `1.` + 空格 | 有序列表 |
| `` ``` `` | 代码块 |
| `---` | 分割线 |

**划词引用**

选中一段文字点击工具栏中「 " 」，可以将文字添加至 chat 输入框中。

**划词评论**

选中一段文字，点击「💬」。

momo 可以读取文档里的所有评论，并在你的授权下回应它们。

**在编辑模式下划词编辑**

选中文字会展示编辑工具栏，可以编辑文本类型（标题、正文、加粗、斜体等）。

**「/」添加内容**

选中文字后，工具栏中点击「/」可在选中位置插入新的内容块，快速在段落间添加文字、列表、代码块等。

## 几个省时间的用法

**让 momo 在当前文档里直接操作**

你打开一篇文档，对 momo 说：「在这篇文档的第二节后面，插入一个关于竞品对比的表格，数据用 Workspace 里已有的竞品分析内容。」

它会直接修改这篇文档，不需要你复制粘贴。

**用 @ 引用文档**

在 momo 的输入框里输入 `@`，可以搜索并引用 Workspace 里的文件。

> "@竞品分析 里关于定价部分的内容，帮我整理成一个对比表。"

也可以直接在左侧**目录树**操作：鼠标悬停在任意文件上，点击文件名旁出现的 `@` 图标，该文件会自动插入当前 chat 输入框，不用手动输入文件名。

**批量处理文件**

> "把「会议纪要」文件夹里所有 2024 年以前的文件，移动到 `_archived/` 文件夹。"

momo 可以一次处理多个文件，不需要你一个个操作。

**设定输出位置**

> "帮我写一份本周工作总结，直接新建一个文件，放到「周报」文件夹，文件名用今天的日期。"

你不需要先建文件再让它写，直接说在哪里创建什么内容就好。

## 编辑与保存

点击文档右上角的「编辑」按钮进入编辑模式，编辑完成后点击右上角的「✓」保存文档。

文档有版本历史，可以通过右上角的「历史版本」查看和恢复任意时间点的内容。
