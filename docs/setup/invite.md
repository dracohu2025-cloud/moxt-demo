# 邀请你的同事一起来

Moxt 是一个团队工具。你一个人用，能感受到它一部分的价值。但当你的同事也在里面工作，AI 能看到所有人的上下文——那才是它真正变强的时候。

发出第一封邀请，是你们团队 AI 飞轮开始转动的那一刻。

---

## 怎么邀请

1. 点击左侧边栏顶部的下拉菜单
2. 选择「设置」→ 「成员」→「邀请成员」
3. 输入同事的邮箱
4. 发送邀请

对方会收到一封邮件，点击链接后直接进入你们的团队工作空间。

你还可以通过「设置」→ 「成员」→「复制邀请链接」，将链接发送给群组或任意想要邀请的同事，对方访问链接点击加入就可以进入你们的团队工作空间。

---

## 邀请谁先来？

不需要等所有人都准备好再一起来。建议先邀请 1-2 个最常需要协作的同事，让你们先跑起来。

通常最有价值的第一批人：

- **经常一起写文档的人**——他们会立刻感受到 momo 在共同文档上的价值
- **负责整理信息的人**——日报、周报、会议纪要，AI 可以替他们完成大量重复劳动
- **需要经常找你同步进展的人**——空间里有所有更新，他们自己就能找到答案

---

## 邀请之后可以做的第一件事

同事进来之后，让他们先做一件事：**和 momo 打个招呼，告诉它自己是谁**。

可以直接发这句话给他们：

> "打开 Moxt 之后，在右侧找到 momo，发给它这句话：'我是 [你的名字]，负责 [你的职责]，以后我们一起工作。'然后看看它怎么回应你。"

这是最好的第一次体验——不是学会怎么用，而是感受到它把你当一个真实的人在认识。

---

## 关于权限

| 角色 | 权限 |
| ---- | ---- |
| **管理员** | 管理成员、修改工作空间设置、管理积分与付款 |
| **成员** | 访问和编辑工作空间的内容 |

工作空间中的个人空间是私有的——你的同事看不到你的个人文件夹，就像你们各自有一张独立的工作桌。
